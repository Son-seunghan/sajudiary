# 🌐 도메인 구매 → 사이트 연결 가이드

사주로그를 본인 도메인(예: sajulog.com)으로 띄우는 전체 절차입니다.

---

## 📋 전체 흐름

```
[1] 도메인 구매       ← 가비아 / Cloudflare / Namecheap
    ↓
[2] 호스팅 가입       ← Vercel 또는 Netlify
    ↓
[3] 코드 업로드       ← GitHub 통해 자동 배포
    ↓
[4] 도메인 ↔ 호스팅 연결  ← DNS 설정
    ↓
[5] HTTPS 자동 적용    ← Vercel/Netlify가 자동
    ↓
[6] 카카오/토스에 도메인 등록  ← 새 주소로 동의
    ↓
완료! 🎉
```

⏱️ **소요 시간**: 첫 도전 시 1~2시간, 익숙해지면 30분

💰 **비용**: 도메인만 유료 (연 14,000~22,000원)

---

## 1단계 — 도메인 구매

### 추천 도메인 등록 업체

| 업체 | 가격 (.com/년) | 장단점 |
|------|---------------|--------|
| **가비아** (https://www.gabia.com) | 약 17,000원 | 한국어 지원, 결제 편함 |
| **Cloudflare** (https://www.cloudflare.com) | 약 14,000원 | 가장 저렴, 영어 UI |
| **Namecheap** (https://www.namecheap.com) | 약 14,000원 | 영어 UI, 첫해 할인 |

**처음이라면 → 가비아 추천** (한국어, 카드 결제 익숙)

### 가비아에서 도메인 구매하기

1. https://www.gabia.com 접속
2. 검색창에 `sajulog` 입력 → 검색
3. `.com` `.kr` `.co.kr` 등 가능한 옵션 표시
   - **추천**: `.com` (전 세계 표준, 가장 무난)
   - 한국 한정 운영이면 `.kr` (.kr은 약간 비쌈)
4. 원하는 도메인 **장바구니 담기** → 결제
5. 등록자 정보 입력 (실명, 이메일)
6. 결제 완료
7. → 가비아 마이페이지에서 **"My가비아 → 서비스 관리"**에 도메인이 보임

📌 **Tip**: 도메인은 1년 단위로 갱신. 가급적 자동 갱신 ON.

---

## 2단계 — Vercel 가입 + 코드 업로드 (GitHub 경유)

### 2-1. GitHub 가입 (이미 있으면 패스)
1. https://github.com 접속 → Sign up
2. 이메일/비밀번호 입력 → 가입 완료

### 2-2. 코드를 GitHub에 올리기 (Repository 생성)
**방법 A — GitHub 웹사이트에서 직접 업로드 (간단)**
1. GitHub 로그인 → 우측 상단 + → "New repository"
2. Repository name: `sajulog` (자유)
3. **Public** 또는 **Private** 선택 (Public 무료 추천)
4. "Create repository" 클릭
5. 다음 화면에서 "uploading an existing file" 클릭
6. **`분석 사이트 - 코드용` 폴더의 모든 파일**을 드래그 앤 드롭
7. "Commit changes" 클릭 → 업로드 완료

**방법 B — Git 명령어 (개발자용, 익숙하지 않으면 A로)**

### 2-3. Vercel 가입 + 연결
1. https://vercel.com/signup 접속
2. **"Continue with GitHub"** 클릭 (가장 편함)
3. GitHub 계정으로 로그인 → 권한 허용
4. Vercel 대시보드에서 **"Add New... → Project"** 클릭
5. 방금 만든 `sajulog` 저장소 선택 → "Import"
6. 설정 화면 그대로 두고 **"Deploy"** 클릭
7. 1~2분 후 자동 배포 완료
8. → `sajulog.vercel.app` 같은 임시 주소가 발급됨
9. 클릭해서 사이트 작동 확인 ✅

📌 GitHub에 새 코드 업로드 = Vercel이 자동으로 재배포 (편함)

---

## 3단계 — 도메인을 Vercel에 연결

### 3-1. Vercel에 도메인 추가
1. Vercel 프로젝트 → **"Settings → Domains"** 메뉴
2. 입력란에 `sajulog.com` 입력 → **"Add"**
3. Vercel이 DNS 설정 정보를 보여줌:
   ```
   Type: A
   Name: @
   Value: 76.76.21.21
   
   Type: CNAME
   Name: www
   Value: cname.vercel-dns.com
   ```
4. 이 값들을 메모해두기 (또는 화면 그대로 켜두기)

### 3-2. 가비아 DNS에서 설정
1. 가비아 마이페이지 → **"My가비아 → 서비스 관리"**
2. `sajulog.com` 옆 **"DNS 정보"** 클릭
3. **"DNS 관리 툴"** → "DNS 설정" 메뉴
4. 기존 레코드 삭제 후 새 레코드 추가:

| 타입 | 호스트 | 값 |
|------|--------|------|
| **A** | `@` | `76.76.21.21` |
| **CNAME** | `www` | `cname.vercel-dns.com.` |

5. 저장
6. 5분~24시간 기다리기 (보통 30분 내 적용)

### 3-3. 확인
1. 5~10분 후 브라우저에서 `sajulog.com` 접속
2. 우리 사이트가 뜨면 성공! ✅
3. Vercel은 HTTPS 인증서를 자동 발급 → 자물쇠 표시

## 4단계 — 카카오 디벨로퍼스에 새 도메인 등록

도메인이 바뀌면 카카오에도 알려야 합니다.

1. https://developers.kakao.com → 사주로그 앱
2. **앱 → 플랫폼 키 → Web 플랫폼**
   - 사이트 도메인 추가: `https://sajulog.com`
3. **카카오 로그인 → 일반 → Redirect URI**
   - 추가: `https://sajulog.com/login.html`
4. 저장

기존 `http://127.0.0.1:5500` 항목은 그대로 둬도 됨 (개발할 때 필요).

## 5단계 — 토스페이먼츠 도메인 등록 (실서비스 시)

테스트 키는 도메인 등록 불필요. 실서비스 키로 전환할 때:
1. https://app.tosspayments.com → 가맹점 정보
2. 도메인 등록 → `sajulog.com` 추가

## 🚨 자주 발생하는 문제

### "도메인 접속이 안 됨"
- **원인**: DNS 전파 지연
- **해결**: 30분~24시간 기다리기
- 진단: https://dnschecker.org 에서 `sajulog.com` 검색 → 전 세계 전파 확인

### "HTTPS 자물쇠 안 뜸"
- **원인**: Vercel이 SSL 인증서 발급 중
- **해결**: 5~10분 더 기다리기. 그래도 안 되면 Vercel에서 도메인 삭제 후 재등록

### "카카오 로그인이 안 됨"
- **원인**: 카카오에 새 도메인 미등록
- **해결**: 4단계 다시 확인

### "GitHub에 한글 파일명/폴더명 업로드 실패"
- **원인**: 일부 환경에서 한글 인코딩 문제
- **해결**: 폴더 이름을 영문으로 변경 (예: `사주 분석` → `saju-analysis`)
