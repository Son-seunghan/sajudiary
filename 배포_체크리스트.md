# 🚀 사주다이어리 배포 전 체크리스트

배포(서비스 정식 오픈) 전에 **반드시 사용자분이 채워 넣어야 할 항목**과 **확인할 사항**을 정리한 문서입니다.

---

## 🔴 필수 입력 항목 — 정식 서비스 시 법적 의무

### 1. 사업자 정보 입력 (`index.html` 푸터)

`index.html` 파일의 푸터(맨 아래 영역)에 다음 4개 placeholder를 본인 정보로 교체:

```
[대표자명]      → 실제 본인 이름 또는 법인 대표자명
[고객센터 이메일] → 실제 운영하는 이메일 (예: support@sajudiary.com)
[등록번호]      → 사업자등록번호 (예: 123-45-67890)
[신고번호]      → 통신판매업 신고번호 (예: 제2026-서울XX-1234호)
[사업장 주소]   → 사업장 등록 주소
```

> **개인사업자도 통신판매업 신고는 의무**입니다 (전자상거래법). 사업자등록 후 30일 내 신고.
> 신청처: 정부24 또는 관할 시청/구청 → 1~7일 내 발급

### 2. 개인정보 보호책임자 정보 입력 (`privacy.html`)

`privacy.html` 제10조 표 안의 placeholder 교체:
```
[대표자 이름 입력 필요]      → 보통 사업자 본인
[고객센터 이메일 입력 필요]  → 위 1번과 동일
```

### 3. 카카오톡 추천 시 도메인 등록

배포 후 도메인이 정해지면 **카카오 디벨로퍼스**에 등록:
- `https://sajudiary.com` (또는 본인 도메인) → 사이트 도메인 추가
- `https://sajudiary.com/login.html` → Redirect URI 추가

## 🟡 권장 입력 항목

### 4. OG 이미지 (`assets/og-image.svg`)
- 카카오톡으로 공유 시 보이는 썸네일
- 현재 SVG로 생성됨, 필요 시 PNG로 변환해 사용 가능
- 직접 디자인하시려면 1200x630 사이즈 PNG 권장

### 5. 토스페이먼츠 본인 키 교체
현재는 토스 공식 데모 키 사용 중. 본인 키로 교체:
1. https://app.tosspayments.com 가입
2. API 키 → 클라이언트 키 복사
3. `assets/js/config.js` 의 `TOSS_CLIENT_KEY` 교체

### 6. 카카오 JavaScript 키 (이미 설정됨 ✅)
- 현재 키: `cc0a54d5766ad91297c565b0c3e9a589`
- 변경 필요 없음
